# bank-accounts

## description
A UI that access a backend API to manage bank accounts

## getting started
1. install dependencies:
`npm install`
2. in a terminal, run test suite:
`npm test`
3. in another terminal, start app in watch mode:
`npm start`

## dependencies
node 14.16
npm 7.3
